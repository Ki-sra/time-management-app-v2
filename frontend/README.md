Frontend (Vite + React + Tailwind)
----------------------------------

Quick start:
1. Install Node.js (16+) and npm/yarn.
2. Run `npm install` in this folder.
3. Create a `.env` file with `VITE_API_URL=http://localhost:8000/api` (or your backend URL).
4. Run `npm run dev` to start the dev server (default port 5173).
5. The app expects the backend API to be running and accessible.
